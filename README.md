# Detox — A 7 Day Fresh Start v13

This build adds a six-step Day 0 setup wizard and a structured food progression:

- Days 1–3: solid fruit-and-vegetable meals
- Day 4: filling smoothies and blendable/substantial soups; not a juice fast
- Days 5–7: a flexible mix of solid meals, soups and smoothies

The suggested food plan is generated from foods, herbs and spices selected during setup. Users may also enter foods to exclude. The shopping checklist is generated from the recipes used in the actual seven-day suggestion.

Other changes include next-day phone/sleep/movement briefings, missed-day recovery, previewable future days, four-action completion scoring and hours/minutes phone-use entry.

All suggestions are flexible. Users should repeat meals, use leftovers, change portions or choose solid food whenever needed.
